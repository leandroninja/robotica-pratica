Sobre o Projeto

Um estudo prático e aprofundado sobre robótica, eletrônica e programação de sistemas embarcados.

Motivação

Por que Robótica?

A robótica é uma das áreas de maior crescimento no mundo tecnológico atual. Combinando hardware e software, permite criar soluções que impactam diretamente a vida das pessoas — desde automação industrial até assistência médica.

Este projeto nasceu como um estudo prático dos fundamentos da robótica, cobrindo eletrônica básica, programação de microcontroladores e integração de sensores e atuadores em projetos reais.

Tecnologias Utilizadas

Ferramentas e plataformas que compõem o ecossistema de estudo.

🔵

Arduino

Plataforma de prototipagem open-source com IDE própria e linguagem baseada em C++.

🟣

Raspberry Pi

Computador de placa única que roda Linux e suporta Python, ideal para projetos mais complexos.

🟡

Python

Linguagem principal para scripts de automação, visão computacional e integração com IA.

🟠

C++

Linguagem de baixo nível usada em firmware de microcontroladores e sistemas embarcados.

🔴

ROS

Robot Operating System — framework de software para desenvolvimento de robôs complexos.

🟢

ESP32

Microcontrolador com Wi-Fi e Bluetooth integrados, ideal para projetos IoT.
